# HateSpeechDetection

Goal of this project is to classify and filter social media posts on hate speech using text analytics methods. 

## Categories

- Bullying
- Harassment
- Racism
- Sexism

## Datasets and papers

- Cyberbullying Datasets (WoW Forum and LoL Forum)
